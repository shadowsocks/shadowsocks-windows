# Experimental Features

Features list here are not fully tested.

#### Listening on local IPv6 interface (v4.1.7)

In `gui-config.json` file, add `isIpv6Enabled` key-value. Default value is `false`.
```
  ...
  "isDefault": false,
  "isIPv6Enabled": true,
  "localPort": 8388,
  ...
```
